Este programa em php é o início da busca em organizar informações climáticas
coletadas em trabalho de campo em Climatologia 01.

Também é um treino na linguagem PHP.

-Foi criado um arquivo "Index.php" para organizar as informações
-Cada ponto de observação possui um arquivo próprio para a coleta de dados "ponto01, ponto02 ...".
-Há o arquivo "retorno.php" que mostra as informações coletadas.

-------------------------------------------------------------------------------------------------

Comandos de treino aplicados:
-echo
-include
-readline(">> ") leitor de Comandos
-print_r() para organizar array.
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Como usar?
-Baixe a pasta do projeto
-No dash abra a pasta (cd /home/user/meus projetos/campo_clima/)
-Execute o arquivo "index.php" (php index.php)
-Informe as variáveis para o "readline(">> ")"
-Por ultimo receba um array com os dados coletados
